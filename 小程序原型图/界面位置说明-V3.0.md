**入口**：首页登录按钮 / 个人中心退出后

## 11. ★ OpenMAIC AI课程模块（V3.0新增，6页）

### 技术架构说明

```
竞赛通服务器（同一台）
├── Spring Boot :8080（竞赛通后端）
│   ├── /api/v1/maic/*（AI课程接口）
│   └── /api/v1/maic/callback（OpenMAIC Webhook回调）
├── OpenMAIC :3100（Next.js，独立进程）
│   └── /api/generate（课件生成）
│   └── /api/sessions（SSO会话管理）
└── Nginx :443
    ├── /api/* → :8080
    └── /maic/* → :3100（AI课堂）
```

SSO单点登录流程：
```
① 竞赛通后端签发 JWT SSO Token（Redis，2h TTL）
② 前端带 Token 跳转 /maic/{classroomId}?token={ssoToken}
③ OpenMAIC 验签，建立课堂会话
④ 课堂结束 → Webhook POST /api/v1/maic/callback
⑤ 同步学习进度到 jst_maic_learn_session 表
```

## 14. 开发规范

### 移动端强制规范

| 规范项 | 要求 |
|--------|------|
| input font-size | ≥ 16px（防 iOS 自动缩放）|
| 底部导航安全区 | `calc(68px + env(safe-area-inset-bottom))` |
| 有底部导航的页面 | `body class="has-tabbar"` |
| fixed 元素 | 加 `transform: translateZ(0)` 防 iOS 抖动 |
| 横滑列表 | **禁用负 margin**，使用 `padding-left/right: var(--space-4)` |
| 最后卡片 | `.card:last-child { margin-right: 0; }` 防截断 |

### OpenMAIC 集成开发规范（V3.0新增）

| 规范项 | 要求 |
|--------|------|
| 进入课堂 | 前端调用后端 `/api/v1/maic/courses/{id}/enter`，禁止直接跳转 OpenMAIC |
| SSO Token | JWT，2小时有效，Redis存储，由后端签发，前端不可伪造 |
| Webhook 回调 | OpenMAIC 回调后端 `/api/v1/maic/callback`，更新 `jst_maic_learn_session` |
| Nginx 配置 | `/maic/*` 需支持 WebSocket：`proxy_http_version 1.1; proxy_set_header Upgrade $http_upgrade; proxy_set_header Connection "upgrade"` |
| LLM配置 | application.yml 中配置 `DASHSCOPE_API_KEY`（通义千问），禁止硬编码 |

### 管理员端开发规范（V3.0新增）

| 规范项 | 要求 |
|--------|------|
| Token | `localStorage.setItem('admin_token', token)`，禁止与用户端混用 |
| 接口鉴权 | 所有 `/system/*` 请求 Header 携带 `Authorization: Bearer {admin_token}` |
| 权限控制 | 菜单列表由 `/system/menu/treeselect` 动态加载，前端根据角色过滤 |
| 会话超时 | 监听 `mousemove/keydown/touchstart`，30分钟无操作调用登出接口 |
| 图片加载 | 凭证/证照图片使用 OSS 临时签名 URL（后端生成，2小时有效），禁止永久URL |
| 操作日志 | 所有审核/发布/封禁操作由 RuoYi AOP 自动写入 `sys_oper_log` |
