# 竞赛通 H5 原型 · 界面位置说明 V4.0

> **版本**：V4.0（独立交付版，2026-04-09）
> **规模**：**81 个页面**（V4.0 PRD 要求 79 页全覆盖 + 线下活动扩展 2 页）
> **目标环境**：微信小程序端（学生/家长/渠道方）+ 赛事方 Web 工作台 + 平台 H5 审核端
> **技术栈**：纯 HTML + CSS + 原生 JS，无框架依赖，`design-system.css` 统一设计令牌
> **赛事主题**：文化艺术类（安徒生艺术奖 / 鸟巢杯 / 人民艺术 / 全国书法展演等）
> **V4.0 核心变更**：角色模型重构（teacher→channel）、积分等级体系、渠道返点结算、赛事方入驻工作台、线下活动预约核销、营销权益体系
