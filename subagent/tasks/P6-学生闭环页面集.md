## 涉及页面 (4 张主页 + 1 个核心组件)

### 1. components/jst-form-render/jst-form-render.vue ⭐ 核心组件
**用途**: 通用动态表单渲染器,根据 schema_json 渲染输入控件
**支持字段类型** (与 PRD/F8 一致):
text / textarea / radio / checkbox / select / date / age / number / image / audio / video / file / group / conditional
**props**:
```js
{
  schema: Object,        // 模板 schemaJson
  value: Object,         // 当前表单值 (双向绑定)
  readonly: Boolean      // 详情查看时只读
}
```
**事件**: `@input(key, val)` / `@validate(errors)`
**实现要点**:
- 用 v-for 遍历 schema.fields
- 根据 type 动态 render 不同的输入组件
- 必填项加红色 `*`
- 校验:onSubmit 时返回所有 required 字段缺失列表
- image / file 字段:用 mock URL 占位 (本期不接 OSS)
- conditional 字段:支持 visibleIf 表达式 (简化为 `{ key, value }`)

**复杂度**: 这是本任务**最难**的部分,允许子 Agent 简化实现 (跳过 group/conditional 复杂类型,标 TODO)

### 2. pages-sub/contest/enroll.vue ⭐ 报名页
**对应 PNG**: enroll.png
**进入路径**: contest/detail.vue 点"立即报名" → 跳本页 + 携带 contestId
**数据流**:
```
onLoad(query.contestId) →
  并行调:
    - getContestDetail(contestId)
    - getEnrollTemplate(contestId)
    - getMyParticipants() (供选择参赛人)
  →
  渲染:
    - 顶部赛事信息卡 (可折叠)
    - 参赛人选择 (radio,如无则跳 my/participant 页认领)
    - jst-form-render 渲染动态表单
  →
  用户填表 →
    - 点"暂存草稿" → POST /jst/wx/enroll/draft
    - 点"立即提交" → 校验所有 required → POST /jst/wx/enroll/submit
  →
  提交成功 → toast + uni.redirectTo /pages-sub/my/enroll-detail?id=enrollId
                (注: 报名成功**不**直接进支付,等 C2 + P7)
```

### 4. pages-sub/my/enroll-detail.vue (报名详情)
**用途**: 看单个报名的完整信息 (用 jst-form-render readonly 模式渲染)
**数据**: GET /jst/wx/enroll/{id}
**结构**:
- 顶部 audit_status 大徽章 + audit_remark
- 赛事信息卡
- 参赛人信息卡 (脱敏 mobile)
- jst-form-render readonly 模式回显表单值
- 底部按钮:
  - audit_status=supplement → 显示"补充材料"按钮 → 跳回 enroll.vue 编辑模式
  - audit_status=approved → 显示"前往支付"按钮 (本期 toast "C2 完成后开放")
  - 其他 → 无按钮

### 5. pages-sub/my/binding.vue (我的绑定)
**对应 PNG**: my-binding.png / binding.png
**数据**: GET /jst/wx/user/binding
**结构**:
- 顶部当前绑定卡片 (如有 active 绑定):
  - 老师/机构头像 (默认占位)
  - 名称
  - 绑定时间
  - "解绑" 按钮 → 二次确认 → POST /jst/wx/user/binding/unbind
- 历史绑定列表 (status != active):
  - 灰度卡片 + status 徽章
- 底部"绑定新老师"按钮 → 跳一个简单的渠道选择页 (输入 channelId 或扫码,本期可简化为 input 输入)
  - 输入 channelId → POST /jst/wx/user/binding/switch

### api/enroll.js (新建)
```javascript
import request from './request'

export const getEnrollTemplate = (contestId) =>
  request({ url: '/jst/wx/enroll/template', method: 'GET', data: { contestId } })

export const saveEnrollDraft = (data) =>
  request({ url: '/jst/wx/enroll/draft', method: 'POST', data })

  request({ url: '/jst/wx/enroll/submit', method: 'POST', data })

export const getEnrollDetail = (id) =>
  request({ url: `/jst/wx/enroll/${id}`, method: 'GET' })

  request({ url: `/jst/wx/enroll/${id}/supplement`, method: 'POST', data })
```
