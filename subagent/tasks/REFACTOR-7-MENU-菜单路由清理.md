## 一、背景

## 二、改造方案

### 2.1 菜单重命名

```sql
-- "竞赛通管理" → "平台数据管理"
UPDATE sys_menu SET menu_name = '平台数据管理' WHERE menu_id = 9800;
-- "赛事方工作台" → "赛事管理中心"
UPDATE sys_menu SET menu_name = '赛事管理中心' WHERE menu_id = 9700;
```
