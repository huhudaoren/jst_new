## C. 实现说明
1. `jst-form-render` 已支持：
   `text / textarea / radio / checkbox / select / date / age / number / image / audio / video / file`
   `group / conditional` 做了简化渲染，支持基础 `visibleIf={ key, value }`。

2. 报名页实现：
   支持选参赛人、动态表单填写、草稿保存、正式提交、补件提交。
   图片/文件类字段用 mock URL 占位，不走真实上传。

3. 我的报名实现：
   顶部审核状态切换。
   卡片展示赛事名、参赛人、材料状态、提交时间。
   待补充状态可直达补件页。

4. 报名详情实现：
   顶部状态卡 + 审核说明。
   赛事信息、参赛人信息、表单只读回显、附件列表。
   `supplement` 显示“补充材料”，`approved` 显示“前往支付”占位 toast。

5. 我的绑定实现：
   有 active 绑定时显示 hero 卡和解绑/更换按钮。
   无绑定时显示空状态。
   更换渠道方采用输入 `channelId` 的简化交互。

6. 我的课程实现：
   顶部 `全部 / 学习中 / 已完成 / AI课堂`。
   学习中和已完成卡片区分蓝绿进度条。
