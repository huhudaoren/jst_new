## 2. 项目背景

"竞赛通"定位为面向学生、家长、老师、机构及赛事组织方的竞赛服务平台，提供一站式的赛事信息查询、报名、进度跟踪、成绩查询、证书查看与下载等服务。

**当前运营方向：文化艺术类赛事**，代表性赛事包括：
- **安徒生(国际)艺术奖**（国际级，主打赛事，¥180/人）
- **"鸟巢杯"文化艺术活动**（全国级，线下，¥150/人）
- **人民艺术全国展演活动**（全国级，免费，线上+线下）

同时配套提供艺术类课程（儿童水彩、素描速写、声乐表演、书法国画等）。

---

## 3. 产品目标

### 3.1 业务目标
- 建立统一的文化艺术赛事信息展示与报名入口，提升报名转化率。
- 沉淀用户、报名、订单、成绩、证书等核心数据。
- 支撑学生自主报名、老师代报名、机构代报名等业务模式。
- 扩展课程服务模块，提供从"赛事"到"学习"的完整链路。

### 3.2 用户目标
- 快速找到适合自己的文化艺术竞赛（按类别/学段/状态筛选）。
- 在手机端顺畅完成报名、缴费、资料提交、进度查询。
- 方便查看赛事通知、成绩、证书与历史报名记录。
- 老师/机构统一管理学生报名并完成批量支付，查看报名数据统计。
- 浏览和学习平台提供的艺术课程内容。

### 3.3 一期目标
- 完成移动端闭环：浏览赛事 → 查看详情 → 报名提交 → 微信支付/对公转账 → 审核确认 → 结果查询。
- 支持基础个人中心（**学生/老师双视角 Tab**）、消息通知、证书下载、证书验真。
- 支持艺术赛事差异化表单（参赛类别）与老师/机构代报名模式，含订单详情子页与报名数据统计页。
- 课程模块支持视频播放、笔记记录、倍速控制、章节切换。

## 4. 用户角色

### 4.1 C 端用户
- **学生**：浏览赛事、自主报名、查看进度、查询成绩、查看下载证书、学习课程、管理与老师的绑定关系。
- **家长**：协助学生完成报名、缴费、资料上传与进度跟踪。

### 4.2 业务角色
- **老师**：绑定学生（一对一）、为学生代报名、查看**订单详情**（学生信息全字段+凭证）、统一支付、查看**报名数据统计**。
- **机构**：管理旗下老师及学生（层级管理）、批量代报名、统一支付、查看报名数据。
- 【重要】老师与机构共用同一套 H5 界面，通过**个人中心双视角 Tab**（`🎓 学生` / `👨‍🏫 老师/机构`）统一管理。

### 4.3 平台角色
- **平台运营**：配置首页、赛事分类、资讯公告、报名规则、审核流程、课程内容。
- **赛事组织方**：发布赛事、管理报名、审核资料、发布成绩与证书。
- **财务/客服**：处理支付对账、退款、异常订单、用户咨询。

## 5. 使用场景

1. 学生或家长想找近期文化艺术竞赛，按**艺术/音乐/舞蹈/美术/朗诵**等分类筛选。
2. 用户查看**安徒生(国际)艺术奖**详情，确认参赛类别（绘画/书法/摄影/手工）、组别（幼/小/初/高）后完成报名，提交作品电子稿。
3. 老师生成**绑定二维码**，学生扫码完成一对一绑定，由老师统一代报名支付（支持对公转账）。
4. 老师通过**「订单详情」页**查看学生报名信息全字段，查看/重新上传对公转账凭证，跟踪订单进度时间轴。
5. 老师在**「报名数据统计」页**查看 KPI 数据、学科分布、热门赛事排行、获奖情况数据看板。
6. 赛后用户查询成绩、下载图片/PDF 格式证书，或扫描证书二维码跳转**公开验真页**。
7. 用户浏览艺术课程（儿童水彩/素描速写/声乐表演/书法国画），用笔记功能记录学习内容。

## 6. 产品定位与范围

### 6.1 产品定位
竞赛通是以手机端 H5 为主的竞赛信息与报名服务平台，当前以**文化艺术类赛事**为主要运营方向，提供"找赛事 → 报赛事 → 查进度 → 查成绩 → 看证书 → 学课程"的完整服务闭环。

### 6.2 一期范围（全部完成原型，共 33 个页面）

**公共入口**（3页）：首页 / 综合搜索 / 登录注册

**赛事主流程**（3页）：赛事列表 / 赛事详情 / 报名流程

**课程模块**（3页）：课程列表 / 课程详情 / 课程播放（含笔记/倍速）

**公告资讯**（3页）：公告列表 / 公告详情（独立页）/ 消息通知中心

**个人中心**（8页）：个人中心（双视角）/ 我的报名 / 我的订单 / 我的成绩 / 我的证书 / 我的课程 / 绑定关系（学生视角）/ 我的消息（重定向）

**账号设置**（4页）：个人资料编辑 / 隐私设置 / 绑定关系管理 / 客服帮助

**公开查询**（2页）：成绩公开查询 / 证书验真（扫码直达）

**老师/机构**（7页）：身份认证 / 老师工作台 / 学生管理 / **老师订单管理** / **订单详情子页** / **报名数据统计** / 机构工作台

### 6.3 明确不做（一期）
- 推广模块（分销、拉新返佣、海报推广等）
- 复杂社交能力（社区、动态、评论互动）
- 重运营商城能力（积分商城）
- 地区分站功能（统一全国站模式）

## 7. 核心流程

### 7.1 学生自主报名流程
```
首页（Banner/推荐赛事）
 → 赛事列表（按艺术/音乐/舞蹈等分类筛选）
 → 赛事详情（安徒生艺术奖等）
 → 查看参赛类别/组别/时间/费用/奖项
 → 登录账号
 → 报名流程（填信息→选类别→上传作品→微信支付）
 → 等待评审
 → 我的报名（查进度）→ 我的成绩 → 我的证书（下载/验真）
```

### 7.2 老师/机构代报名流程
```
个人中心 → 老师视角 Tab → 完成身份认证
 → 老师工作台 → 生成绑定二维码
 → 学生扫码绑定（或手动输入绑定码）
 → 学生管理 → 选择学生 → 代报名（进入赛事列表）
 → 填写学生报名信息
 → 统一支付（微信/对公转账）
 → 订单详情（查看学生信息/凭证/进度时间轴）
 → 报名数据统计（整体数据看板）
```

### 7.3 证书查验流程
```
我的证书 → 查看证书 → 下载（JPG/PDF）
           → 验真 → 跳转 cert-verify.html（公开页，无需登录）
```

## 8. 信息架构

### 8.1 底部 Tab 导航（5项）
`🏠 首页` / `🏆 赛事` / `📚 课程` / `📢 公告` / `👤 我的`

### 8.2 首页结构（文化艺术主题）
- 顶部品牌区：Logo + 搜索框（→ search.html）+ 消息入口 + 登录按钮
- Banner 轮播（3张，紫/红/蓝渐变主题）：安徒生艺术奖 / 鸟巢杯 / 人民艺术
- 6宫格快捷入口：热门赛事 / 赛事分类 / 报名查询 / 成绩查询 / 证书查询 / 课程入口
- 推荐赛事横滑（4卡，`padding-left/right` 控制边距，无负 margin）
- 精选赛事纵排（3条）
- 推荐课程横滑（4卡：水彩/素描/声乐/书法）
- 最新公告
- 平台数据介绍
- 悬浮客服按钮

### 8.3 个人中心结构（双视角 Tab）

```
my.html
├── 🎓 学生视角（默认）
│   ├── 统计数据4格（报名/成绩/证书/课程）
│   ├── 绑定老师状态卡（已绑/未绑双态）
│   ├── 我的服务（报名/订单/成绩/证书/课程）
│   ├── 消息与互动（消息通知/绑定关系）
│   └── 帮助与设置（个人资料/客服/隐私/关于）
│
└── 👨‍🏫 老师/机构视角
    ├── 统计数据4格（绑定学生/代报名总数/待支付/累计支付）
    ├── 认证状态三态区（未认证→审核中→已认证）
    ├── 已认证：工作台大入口卡 + 3格快捷（学生管理/二维码/统一支付）
    ├── 老师功能（学生管理/代报名/统一支付/报名数据/学生成绩/学生证书/机构工作台）
    └── 消息与设置（消息/认证信息/客服/隐私）
```

**切换逻辑**：`switchView('student'|'teacher')`，Hero 主题色随视角切换（学生=品牌蓝，老师=深靛蓝）。URL 参数 `?view=teacher` 可直接展示老师视角。

## 11. 非功能需求

- 首页首屏加载时间 ≤ 3 秒。
- 列表分页加载响应 ≤ 2 秒。
- 高峰期支持并发报名，避免重复支付和超额报名。
- 个人信息加密存储与脱敏展示（手机号中间4位，身份证号脱敏）。
- 优先适配微信内置浏览器（WebView），同时支持 Safari/Chrome。
- 证书下载生成 ≤ 10 秒。
- **横滑列表使用 `padding-left/right` 控制边距，禁止使用负 margin**，确保首尾卡片完整显示。
- input/textarea 的 font-size ≥ 16px，防止 iOS 自动缩放。
