## 一、快速入门

### 1.1 如何打开小程序

在微信中搜索「竞赛通」或扫描二维码进入小程序。首次进入会自动跳转到登录页面。

### 1.2 登录与注册

1. 点击「微信快捷登录」按钮
2. 授权微信登录（自动获取 openid）
3. 首次登录需绑定手机号：点击「绑定手机号」按钮，授权微信获取手机号
4. 登录成功后自动跳转到首页

> 注意：手机号绑定后不可更改，请确认手机号正确。

### 1.3 底部导航栏

小程序底部有 5 个主入口：

| 图标 | 名称 | 功能 |
|---|---|---|
| 🏠 | 首页 | 推荐赛事、热门课程、公告、快捷入口 |
| 🏆 | 赛事 | 赛事列表、分类筛选、搜索 |
| 📚 | 课程 | 课程列表、分类浏览 |
| 📢 | 公告 | 平台公告与资讯 |
| 👤 | 我的 | 个人中心、订单、积分、设置 |

首页从上到下依次展示：

## 八、报名与支付流程

### 8.1 完整报名流程

```
赛事详情 → 点击"立即报名" → 选择参赛人档案 → 填写报名表单 → 提交报名
→ 等待审核 → 审核通过 → 创建订单 → 选择支付方式 → 完成支付
```

### 8.2 详细步骤

**第 1 步：选择参赛人**
- 从已有档案中选择参赛人
- 或新建参赛人档案（姓名、身份证、学校等）

**第 2 步：填写报名表单**
- 根据赛事方配置的动态表单填写信息
- 支持文本、选择、文件上传等字段类型
- 可保存草稿，稍后继续

**第 3 步：提交报名**
- 确认信息无误后提交
- 提交后状态变为「待审核」

**第 4 步：等待审核**
- 赛事方审核报名信息
- 审核结果：通过 / 驳回 / 需要补充材料
- 驳回后可修改后重新提交

**第 5 步：支付**
- 审核通过后，在「我的报名」中点击「去支付」
- 支持以下支付方式（可混合使用）：
  - 微信支付（现金部分）
  - 积分抵扣（按比例折算）
  - 优惠券（满减/折扣）
- 零元订单直接完成，无需支付

**第 6 步：完成**
- 支付成功后报名生效
- 可在「我的报名」中查看报名详情

### 8.3 注意事项

- 报名截止时间前才能提交
- 一个参赛人同一赛事只能报名一次
- 优惠券在支付时选择，支付成功后自动使用
- 积分在支付时冻结，支付成功后正式消耗

## 九、订单与退款

### 9.1 订单状态说明

| 状态 | 说明 | 可操作 |
|---|---|---|
| 待支付 | 订单已创建，等待支付 | 去支付、取消订单 |
| 已支付 | 支付成功 | 申请退款 |
| 已完成 | 赛事已结束 | 查看详情 |
| 已取消 | 用户主动取消或超时取消 | — |

### 9.2 查看订单

路径：我的 → 我的订单
- 支持按状态筛选
- 点击订单卡片查看详情
- 详情页展示：订单号、赛事信息、参赛人、金额明细、支付方式、时间

### 9.3 申请退款

1. 在订单详情页点击「申请退款」
2. 填写退款原因
3. 提交后等待审核
4. 退款审核流程：赛事方审核 → 平台复核
5. 审核通过后自动原路退回

**退款规则**：
- 本期仅支持全额退款
- 退款顺序：微信现金 → 积分回退 → 优惠券回退
- 优惠券回退条件：原券仍在有效期内
- 售后期限内才可发起退款

### 9.4 退款状态说明

| 状态 | 说明 |
|---|---|
| 待审核 | 已提交，等待赛事方审核 |
| 审核中 | 赛事方已受理 |
| 已退款 | 退款完成，资金已到账 |
| 已驳回 | 退款被拒绝（可查看原因）|
| 已取消 | 用户主动取消退款申请 |

## 十、积分与权益体系

### 10.1 积分中心

路径：我的 → 积分中心

- **可用积分**：当前可使用的积分余额
- **冻结积分**：待支付订单中冻结的积分
- **累计获取**：历史总共获得的积分

### 10.2 积分获取方式

| 途径 | 说明 |
|---|---|
| 报名赛事 | 支付成功后获得积分 |
| 完成课程 | 学习完成后获得积分 |
| 每日签到 | 登录签到获得积分 |
| 平台活动 | 参与活动获得积分 |

### 10.3 积分使用

- 赛事报名时抵扣部分费用
- 积分商城兑换商品
- 权益核销时消耗

### 10.4 等级体系

- 等级由成长值决定（非积分）
- 成长值通过参赛、学习、消费等行为累积
- 等级越高，享受的专属权益越多

### 10.5 优惠券

路径：我的 → 我的优惠券

| 状态 | 说明 |
|---|---|
| 未使用 | 可在支付时选择使用 |
| 已使用 | 已在某笔订单中使用 |
| 已过期 | 超过有效期未使用 |

领券入口：首页 → 我的优惠券 → 领券中心

### 10.6 专属权益

路径：我的 → 专属权益
- 查看持有的权益（课程减免、客服特权等）
- 查看剩余使用次数
- 申请核销权益

## 十一、积分商城

### 11.1 浏览商品

路径：首页宫格「积分商城」或 我的 → 积分商城

- 商品列表展示：封面、名称、积分价格、库存
- 支持搜索和分类筛选

### 11.2 兑换流程

1. 选择商品 → 查看详情
2. 确认积分余额足够
3. 点击「立即兑换」
4. 确认兑换信息
5. 兑换成功

**商品类型**：
- **虚拟商品**：优惠券/权益，兑换后直接到账
- **实物商品**：需要填写收货地址，等待发货

### 11.3 我的兑换

路径：积分商城 → 我的兑换
- 查看兑换订单状态
- 实物商品可查看物流信息

## 十三、渠道方工作台

### 13.1 工作台首页

路径：我的 → 切换渠道方 → 进入我的工作台

工作台展示：
- 本月 KPI：新增学生数、订单数、返点金额、支付金额
- 功能入口：学生管理、订单管理、返点中心、提现结算
- 近期订单列表

### 13.2 学生管理

- 查看绑定学生列表
- 查看学生的成绩和证书
- 管理临时参赛档案

### 13.3 渠道订单

- 查看通过本渠道的订单列表
- 查看订单详情（金额、状态、返点）

### 13.4 返点中心

返点来源：学生通过渠道报名并支付成功后，渠道方获得对应比例的返点收入。

- **7 个 Tab 视图**：
  - 可提现：已结算可提现的返点
  - 审核中：提现申请待审核
  - 已打款：已提现到账
  - 已回退：因退款回退的返点
  - 全部：所有返点记录
  - 结算中：正在结算的返点
  - 冻结中：暂时冻结的返点

### 13.5 提现结算

1. 查看可提现金额
2. 选择提现的返点明细
3. 填写银行账户信息
4. 提交提现申请
5. 等待平台审核和打款

### 13.6 经营分析

- 代报名总数统计
- 支付金额统计
- 退款金额统计
- 返点金额统计
- 按赛事/学生维度排行

### 13.7 批量报名

渠道方可以为未注册的学生批量报名：
1. 进入批量报名页面
2. 上传学生名单（Excel 格式）
3. 系统自动创建临时参赛档案
4. 确认信息后批量提交报名

### 13.8 团队预约

渠道方可以创建团队预约：
1. 选择赛事和预约时段
2. 添加团队成员
3. 提交团队预约
4. 查看团队签到状态

## 十四、合同与发票

### 14.1 合同中心

路径：我的 → 渠道方 → 基础管理 → 合同中心

- 查看与平台签订的合同列表
- 按状态筛选：全部/待签署/已签署/已到期
- 点击查看合同详情
- 已签署合同可下载查看 PDF 文件

### 14.2 开票中心

路径：我的 → 渠道方 → 基础管理 → 开票中心

- 查看发票列表和状态
- 按状态筛选：全部/待审核/开票中/已开票

### 14.3 申请开票

1. 点击「申请开票」
2. 选择关联的已打款提现记录
3. 填写发票信息：
   - 发票抬头（必填）
   - 税号（必填，15-20 位）
   - 金额（自动填充）
   - 备注（选填）
4. 提交申请
5. 等待平台审核和开票

## 十六、常见问题

### Q1：报名提交后可以修改吗？
A：待审核状态下不可修改。如被驳回或要求补充材料，可修改后重新提交。

### Q2：支付失败怎么办？
A：订单在待支付状态下可以重新发起支付。超时未支付的订单会自动取消。

### Q3：退款多久到账？
A：审核通过后，微信支付部分 1-3 个工作日到账。积分和优惠券即时回退。

### Q4：积分过期吗？
A：积分不会过期，但冻结的积分在订单取消后会自动解冻。

### Q5：如何切换学生/渠道方视角？
A：渠道认证通过后，在「我的」页面顶部会出现「学生/渠道方」切换按钮。

### Q6：渠道返点什么时候可以提现？
A：学生支付成功后产生返点预计算，结算完成后可提现。退款会导致返点回退。

### Q7：忘记密码怎么办？
A：小程序使用微信授权登录，无需密码。手机号通过微信授权获取。

### Q8：如何联系客服？
A：我的 → 帮助与设置 → 客服与帮助。
